# Requirements
##

Installing Graphql requires two plugins.
If none of them already exist, you can create one by executing the command:

**Backend-Engine Plugin Instance**

```sh
$ node glue add backend-engine engine
```

**Postgres Plugin Instance**

```sh
$ node glue add postgres postgres
```
